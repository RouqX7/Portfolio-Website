# Firebase Configuration Template
# Copy this content to a new file called .env and fill in your Firebase project details

VITE_FIREBASE_API_KEY=your_api_key_here
VITE_FIREBASE_AUTH_DOMAIN=your_project_id.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project_id.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
VITE_FIREBASE_APP_ID=your_app_id

# Instructions:
# 1. Go to your Firebase Console: https://console.firebase.google.com/
# 2. Select your portfolio project (portfolio-website-cc60d)
# 3. Click on the gear icon (Project Settings)
# 4. Scroll down to "Your apps" section
# 5. If you don't have a web app, click "Add app" and choose Web
# 6. Copy the config values from the firebaseConfig object
# 7. Paste them in your .env file (without quotes)
# 8. Make sure to create the file as .env (not .env.txt)
